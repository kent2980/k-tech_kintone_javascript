# PLダッシュボード モバイル版

PLダッシュボードのモバイルデバイス専用バージョンです。

## 特徴

### 🚀 **モバイル最適化**

- スマートフォン・タブレット向けUI
- タッチフレンドリーな操作感
- レスポンシブデザイン対応

### 📱 **モバイル専用機能**

- **カード表示**: データをカード形式で見やすく表示
- **サマリーカード**: 期間統計を一目で把握
- **タッチ操作**: スワイプ・タップ最適化
- **軽量化**: モバイル向けにパフォーマンス最適化

### 🎯 **自動検出**

- モバイルデバイス自動検出
- 画面サイズによる適応表示
- デスクトップ版との共存可能

## ビルド・デプロイ

### モバイル版のみビルド

```bash
npm run build:mobile
```

### モバイル版デプロイ

```bash
npm run deploy:app43-mobile
```

### 両バージョン同時ビルド

```bash
npm run build
```

## ファイル構造

```
src/BOX/PL_dashboard/
├── PL_dashboard.ts          # デスクトップ版
├── PL_dashboard.mobile.ts   # モバイル版 ⭐
├── PL_dashboard.css         # デスクトップ版CSS
├── PL_dashboard.mobile.css  # モバイル版CSS ⭐
└── components/                  # 共通コンポーネント
```

## 使用方法

### 1. モバイルアクセス

スマートフォン・タブレットでkintoneアプリにアクセスすると自動的にモバイル版が起動します。

### 2. フィルター操作

- **年・月選択**: タップしやすいドロップダウンメニュー
- **自動更新**: 選択変更時に自動でデータ更新

### 3. データ表示

- **サマリーカード**: 期間集計を色分け表示
- **詳細カード**: 各レコードをカード形式で表示
- **スクロール**: なめらかなタッチスクロール対応

## 技術仕様

### 📱 対応デバイス

- **iOS**: Safari 13+
- **Android**: Chrome 90+
- **画面サイズ**: 375px〜768px

### 🎨 デザインシステム

- **カラーパレット**: Material Design準拠
- **タイポグラフィ**: Noto Sans JP
- **アニメーション**: CSS transitions
- **レスポンシブ**: CSS Grid & Flexbox

### ⚡ パフォーマンス

- **バンドルサイズ**: 約299KB（gzip: 93KB）
- **レンダリング**: 最適化済み
- **スクロール**: ハードウェアアクセラレーション

## デバッグ・開発

```javascript

/* カスタムカラーの例 */

}
```

## トラブルシューティング

### ❌ モバイル版が表示されない

- ユーザーエージェントを確認
- 画面幅が768px以下かチェック
- コンソールエラーを確認

### ⚠️ データが読み込まれない

- ネットワーク接続を確認
- kintoneアクセス権限をチェック
- エラーメッセージを確認

### 🐛 レイアウトが崩れる

- CSSファイルの読み込み状況を確認
- ブラウザキャッシュをクリア
- デベロッパーツールで要素検証
